# Development Guide

## Extension Development

```bash
cd extension
# Load in Chrome as unpacked extension
# Open chrome://extensions/, enable Developer mode, click "Load unpacked"
```

## Local Server Development  

```bash
cd local-server
npm run dev
```

## Testing

- React: localhost:3000
- Vite: localhost:5173  
- Next.js: localhost:3000
